POST/api/chat

调用Agent完成普通问答、完整条件推荐或已确认购买记录查询。

请求字段

          

            
字段类型必填说明

            
              
user_idstring是用户隔离标识，当前由客户端提供。

              
session_idstring是会话标识，与user_id组成Graph线程键。

              
messagestring是当前用户输入。

              
frontend_selectionobject/null否品类、预算、关注点、避免项及确认状态。

              
historyarray否调用方维护的最近对话。

            
          

        


        

          
响应字段

          

            
字段类型说明

            
              
answerstringAgent最终回答。

              
intentstring/null模型识别的当前问题类型。

              
missing_fieldsstring[]尚未确认的推荐条件。

              
requirement_contextobject当前已确认的结构化需求。

              
required_actionstring/null调用方下一步需要完成的动作。

              
action_optionsarray缺失条件对应的可选项。

              
recommended_productsarray带推荐分和证据状态的商品。

              
comparison_tablearray候选商品对比结构。

              
review_evidencearrayChroma召回并经BGE精排的评论证据。

              
risk_notesstring[]历史数据、证据和推荐边界。

              
raw_responseobjectAgent原始结构，供开发调试。

            
          

        


        

          
当前限制

          

            当前FastAPI尚未提供Graph Resume、确认购买和购买历史管理端点。缺少购买条件的多轮弹窗及购买确认由Streamlit直连Agent完成。接口没有认证、限流和生产级持久化checkpoint，不应直接暴露到公网。
